# Giới Thiệu User Stories

Tài liệu này chứa tất cả User Stories cho hệ thống Chip Định Danh Thú Cưng.

## Cấu Trúc

User Stories được tổ chức theo **vai trò**:

| Folder | Vai trò | Mô tả |
|--------|---------|-------|
| [**ADM**](./ADM/) | Quản trị viên (Admin) | Quản lý kho, nhập/xuất chip, xử lý yêu cầu |
| [**CLI**](./CLI/) | Phòng khám (Clinic) | Lễ tân, bác sĩ, quản lý tồn kho tại clinic |
| [**OWN**](./OWN/) | Chủ nuôi (Owner) | Tạo hẹn, định danh bản thân, tìm hiểu chip |
| [**MER**](./MER/) | Merchant | (Chưa có US) |
